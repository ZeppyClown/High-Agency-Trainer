# 24-Week Skill Curriculum

## Phase 1 (Weeks 1-4)
- Learn macOS permissions model (TCC, Accessibility, Automation).
- Build collector and verify event quality manually.
- Outcome: stable event logging every N seconds.

## Phase 2 (Weeks 5-8)
- Implement deterministic sessionization and replay safety.
- Learn idempotency and queue semantics.
- Outcome: reliable day timeline without AI.

## Phase 3 (Weeks 9-12)
- Add rule-first classification and optional LLM fallback.
- Introduce correction loop for relabeling.
- Outcome: categorized sessions with confidence.

## Phase 4 (Weeks 13-16)
- Build calendar block generation and edit/review UI.
- Outcome: suggested daily/weekly AI calendar.

## Phase 5 (Weeks 17-20)
- Harden app with observability, redaction, and packaging.
- Outcome: installable local application.

## Phase 6 (Weeks 21-24)
- Run weekly architecture review and benchmark loops.
- Outcome: measurable v1.0 with ADRs and tests.
